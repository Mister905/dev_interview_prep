SQL Interview Practice - Questions Only

Practice these challenges using W3Schools SQL Tryit Editor.

Sample Database

Tables:

employees(id, name, department_id, salary, hire_date)
departments(id, name, location)
projects(id, name, department_id, budget)

Use the following sample data:

employees:
1, Alice, 1, 70000, 2020-01-15
2, Bob, 2, 50000, 2019-03-22
3, Carol, 1, 80000, 2021-07-01
4, David, 3, 55000, 2022-02-12
5, Eve, 2, 60000, 2020-11-30

departments:
1, Engineering, NY
2, HR, SF
3, Marketing, LA

projects:
1, Project A, 1, 100000
2, Project B, 2, 50000
3, Project C, 3, 75000
4, Project D, 1, 200000
SQL Practice Problems

1. Select All Employees

Return all columns for all employees.

2. Employees in Engineering

Return all employees whose department_id is 1.

3. Employees with Salary > 60000

Return name and salary for employees making more than 60000.

4. Order Employees by Hire Date

Return name and hire_date ordered by earliest hire date first.

5. Count Total Employees

Return the total number of employees as a single number.

6. Average Salary per Department

Return department_id and average salary of employees in that department.

7. Highest Salary Employee

Return the name and salary of the employee with the highest salary.

8. Employees Hired After 2020-01-01

Return all columns for employees hired after 2020-01-01.

9. Join Employees with Departments

Return employee name and department name by joining employees with departments.

10. Employees in NY Department

Return employee names where the department location is 'NY'.

11. Departments with More Than 1 Employee

Return department name and number of employees for departments with more than 1 employee.

12. Employees in Projects with Budget > 100000

Return employee name and project name where project budget is greater than 100000.

Assume employees are in the same department as projects.

13. Employees Earning More Than Department Average

Return name and salary of employees earning more than the average salary in their department.

14. Total Budget per Department

Return department_id and total budget of all its projects.

15. Employee Names Concatenated with Department Names

Return results like: "Alice - Engineering".
